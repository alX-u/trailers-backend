# Setting up

1. `yarn install` to download dependencies
2. Download docker
3. Remember to set-up .env files. (see template)
4. To launch the local Postgres DB:
   `docker-compose --env-file .env.dev up -d`
5. Launch project in dev mode with:
   `yarn start:dev`
